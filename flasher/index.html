Provision a Tildagon

  1. Power off the badge from the menu, if it's already powered on.
  2. Hold boop while plugging USB into USB in.
  3. Keep holding boop until you click Connect and click on the badge COM port. The device name should appears as USB JTAG/serial debug unit.
    Note: If you see Espressif CDC Device, power off the badge and try these steps again, holding down the boop button for longer.
  4. Let go of the boop.
  5. Select Install Tildagon, and wait.

For more information, see Flash the badge.
